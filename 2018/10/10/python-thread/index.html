python 线程

发表于 2018-10-10 | 分类于 编程

threading 提供线程的相关操作。

threading 模块提供的类 ：
Thread, LOck, Rlock, Condition, Event, Timer, local, [Bounded]Semaphore.

threading 模块提供的常用方法

  • threading.currentThread(); 返回当前的线程数
  • threading.enumerate(); 返回一个正在运行的线程列表。
  • threading.activeCount(); 返回正在运行的线程数。

threading 模块提供的常量
threading.TIMEOUT_MAX 设置 threading 全局超时时间。

Thread 类

Thread 是线程类，使用这个类有两种方法，一个是直接用Thread类实例化一个对象，另外
是继承Thread，然后重写 run 方法。我一般都是直接实例化一个对象来使用Thread。示例代码：

Thread类的构造方法 ：
Thread(group=None,target=None,name=None,args=(),kwargs={})

  • group ：线程组，目前还没有实现，库引用中提示必须是None
  • target：要执行的函数名
  • name ：你要设置的线程名，可以为空。
  • args/kwargs ：函数的参数

Thread 类中的常用方法

  • isAlive(): 返回线程状态
  • getName()/setName(): 获取/设置线程名
  • start(): 线程准备就绪，等待cpu调度
  • isDeamon(bool)/setDeamon(): 获取/设置守护线程
  • join(): 将线程加入到当前线程中。
