socket模块说明

该模块提供套接字操作和一些相关功能。在Unix上，它支持IP（Internet协议）和Unix域套接字。
在其他系统上，它只支持IP。特定于套接字的函数可用作套接字对象的方法。

函数：

socket（） - 创建一个新的套接字对象

socketpair（） - 创建一对新的套接字对象

fromfd（） - 从打开的文件描述符[]

fromshare（）创建一个套接字对象 - 创建来自socket.share（）的数据的套接字对象（）[*]

gethostname（） - 返回当前主机名

gethostbyname（） - 将主机名映射到其IP地址

gethostbyaddr（） - 将IP号或主机名映射到DNS信息

getservbyname（） - 将服务名称和协议名称映射到端口号

getprotobyname（） - 将协议名称（例如’tcp’）映射到数字

ntohs（），ntohl（） - 将16位，32位int从网络到主机字节顺序

htons（），htonl（） - 将16位32位int从主机转换为网络字节顺序

inet_aton（） - 将IP地址字符串（123.45.67.89）转换为32位打包格式

inet_ntoa - 将32位打包格式IP转换为字符串（123.45.67.89）

socket.getdefaulttimeout（） - 获取默认超时值

socket.setdefaulttimeout（） - 设置默认超时值

create_connection（） - connec ts添加到地址，并带有可选的超时和可选的源地址。

[*]在所有平台上都不可用！特殊对象：

SocketType - 用于套接字对象的类型对象

error -为I / O错误引发异常

has_ipv6 - 指示是否支持IPv6的布尔值

IntEnum常量：

AF_INET，AF_UNIX - 套接字域（socket（）调用的第一个参数）

SOCK_STREAM，SOCK_DGRAM，SOCK_RAW - 套接字类型（第二个参数）

整型常量：

可以定义许多其他常量;这些可用于调用setsockopt（）和getsockopt（）方法。
