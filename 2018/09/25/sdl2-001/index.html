SDL2 使用笔记一

sdl2 英文教程：http://www.lazyfoo.net/tutorials/SDL/index.php

SDL2 是外部库，在使用是要导入和配置链接，在 ubuntu 下 使用 code::Blocks 编译SDL项目时要添加
对 SDL 库的链接。

SDL2 的调用分为以下几步：

  1. 初始换 SDL 库，以及一些要用到的配件函数

  2. 将需要的数据加载进内存中，一般是一次性把需要的数据材料加载进内存，后续就不要再进行数据的加载操作了

  3. 事件处理逻辑，和页面渲染

  4. 关闭和释放之前程序初始化的用到的内存资源。
