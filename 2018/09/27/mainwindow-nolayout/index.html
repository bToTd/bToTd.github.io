mainwindow_nolayoutAttempting to add QLayout "" to MainWindow "", which already has a layout

发表于 2018-09-27 | 分类于 qt编程

qt中给QWidget或者QDialog设置布局方法很简单。直接创建一个布局，把想要加的控件往上面
布局上放，最后在QWidget或者QDialog上调用setLayout()就可以了。

然而，我想QMainWindow中使用这种办法来控制界面布局时却出现了下面错误

这个问题又再次展现了网络的厉害，
这个问题，我找到了这个URL：http://www.cnblogs.com/oakentree/p/4433086.html

非常感谢这个博主，他写的这个文章解答了我的疑惑。
QMainWindow这个类已经有默认布局了，而且不能修改，所以呢，要得到你想要的布局，你就不能直接
在QMainWindow上改，具体怎么该，这个博主也给了答案。

修改作出你想要的界面就下面几步：

  1. 创建一个 QWidget 实例，这个的布局我们是可以随意改的
  2. 把你要的布局界面弄到 QWidget 上
  3. 在你的 QMainWindow 实例中 setCentralWidget() 以下
