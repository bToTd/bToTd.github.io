使用模板

发表于 2019-07-25 | 分类于 博客 ， hexo

hexo 是一个非常的好用的个人博客搭建模块。现在一直在使用 hexo 这个工具，但是一直都只是会简单的几个命令，
但是这个用着不怎么爽。好久之前弄过 hexo 的一些配置，但是现在基本都忘了。

文章模板的设置，默认的 post 模板是没有分类的头信息的，但是最近想给自己的文章加上分类，为了更好的管理自己
抄的文章，或者自己写的一些东西。修改配置什么的，最好的方法就是阅读官方的文档。一般都会有，毕竟我不是第一个
使用 hexo 这个工具的，我现在遇到的坑，应该很多人都遇到过，毕竟 hexo 的用户量还是挺多的。

修改文章的模板

官方的连接: https://hexo.io/zh-cn/docs/writing

文档介绍的很详细，就是要修改 Front-matter 的字段。这些字段包括：

参数 描述 默认值
layout 布局
title 标题
date 建立日期 文件建立日期
updated 更新日期 文件更新日期
comments 开启文章的评论功能 true
tags 标签（不适用于分页）
categories 分类（不适用于分页）
permalink 覆盖文章网址

其中，categories 这个分类字段是我想要加上去的。字段是找到了，但是应该在哪个文件上改呢？

在新建文章时，Hexo 会根据 scaffolds 文件夹内相对应的文件来建立文件。也就是说，文章的
模板是放在了 hexo 项目的 scaffolds 文件夹下，直接去这个文件夹下去找要修改的文件就可以了。
不过最好是自己新建一个自己的模板文件。

categories 字段

这个字段后面可以有多个参数，这些参数就像一个个文件夹，这些文件夹是有层级关系的，而不是一个一个
单独没有联系的文件夹。
